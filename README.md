# Sistem Manajemen Perpustakaan Mini (Java)

## Project UTS Pemrograman Berbasis Objek

Nama: Puja widyasti
NIM: 3337240012

---

## Deskripsi
Program ini merupakan sistem manajemen perpustakaan mini berbasis Java yang dibuat untuk memenuhi tugas UTS mata kuliah Pemrograman Berbasis Objek.

Program ini menerapkan konsep Object Oriented Programming (OOP) seperti:
- Class dan Object
- Inheritance
- Encapsulation
- Method dan Method Overloading

---

## Fitur Program
- Registrasi member (nama dan NIM)
- Melihat daftar buku
- Peminjaman buku
- Pengembalian buku
- Perhitungan denda keterlambatan
- Pembatasan jumlah pinjam berdasarkan NIM

---

## Struktur Class
- **Main** → Mengatur alur program
- **User** → Parent class
- **Student** → Mengelola member dan peminjaman
- **Admin** → Menambahkan buku
- **Book** → Data buku
- **Library** → Mengelola daftar buku
- **Transaction** → Menghitung denda

---

## Cara Menjalankan Program
1. Compile semua file:
   javac *.java

2. Jalankan program:
   java Main

Catatan:
Program ini dibuat sebagai simulasi sederhana sistem perpustakaan dan dijalankan melalui console (terminal).
